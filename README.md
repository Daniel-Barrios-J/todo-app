# Todo App

## Scripts para desarrollo una vez clonado el repositorio

### `npm install`

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

## para generar la build

### `npm run build`
